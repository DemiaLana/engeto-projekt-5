# 📊 Analýza mezd, cen potravin a kupní síly v České republice (2000--2025)

Interaktivní **Power BI dashboard**, který porovnává:

-   průměrné mzdy podle odvětví\
-   ceny vybraných potravin\
-   reálnou kupní sílu s ohledem na inflaci potravin

------------------------------------------------------------------------

## 🔎 Hlavní stránky reportu

### 1️⃣ Přehled

Celkový pohled na: - vývoj mezd v ČR\
- ceny potravin\
- kupní sílu pro vybraný produkt

### 2️⃣ Produkty a ceny

-   Vývoj cen vybraných potravin\
-   Meziroční růst (YoY)\
-   Kumulativní změny cen

### 3️⃣ Odvětví & Reálná mzda

-   Reálné mzdy podle odvětví (po zohlednění inflace potravin)\
-   Top 5 odvětví podle reálné mzdy\
-   Meziroční změny

------------------------------------------------------------------------

## 🛠 Použité technologie

-   **Power BI Desktop**
-   **Power Query**
    -   čištění a transformace dat\
    -   Unpivot\
    -   Append\
-   **DAX**
    -   výpočet reálné mzdy\
    -   výpočet kupní síly\
    -   meziroční růst (YoY)

------------------------------------------------------------------------

## 📂 Zdroje dat

-   Český statistický úřad (ČSÚ)
    -   průměrné mzdy podle odvětví\
    -   průměrné ceny vybraných potravin\
    -   index spotřebitelských cen (CPI)
